Importer de l'équipement
Copiez les données exportées dans la zone de texte ci-dessous puis utilisez le bouton 'Recevoir' pour ajouter l'équipement. Note : le bouton dans la ligne de l'équipement, permet de produire le texte à copier ici.
Exporter de l'équipement
L'équipement et les tests associés ont été copiés dans le presse-papier. Pour copier cet équipement, la personne qui va recevoir l'équipement doit cliquer sur "Importer de l'équipement" en bas de l'onglet "Équipement" et coller le text de votre presse-papier.
Vous pouvez copier à nouveau dans le presse-papier le même équipement avec le bouton ci-dessous.
Acheter de l'équipement
Recherchez l'équipement qui vous intéresse, achetez-le avec votre fortune ou ajoutez-le gratuitement (vous pouvez corriger le prix au besoin).
